# Projetos Python — Básico ao Intermediário

Este repositório reúne **5 projetos desenvolvidos em Python puro**, com foco em **lógica de programação, estruturas fundamentais da linguagem e organização de código**.

Os projetos foram implementados para execução em **terminal/console**, sem uso de frameworks ou bibliotecas externas — apenas a biblioteca padrão do Python (como `random` quando necessário).

> Objetivo:  
> Demonstrar domínio dos **fundamentos de Python**, capacidade de estruturar pequenos sistemas e evolução progressiva na resolução de problemas.

## Descrição dos Projetos

### Projeto 1 — Pedra, Papel e Tesoura
Implementação do jogo clássico contra o computador.

**Conceitos aplicados:**
- Estruturas condicionais
- Entrada e saída de dados
- Lógica de decisão
- Uso do módulo `random`

---

### Projeto 2 — ATM (Caixa Eletrônico)
Simulação de um caixa eletrônico com operações básicas.

**Conceitos aplicados:**
- Controle de saldo
- Fluxo de menus
- Validação de operações
- Estruturas condicionais

---

### Projeto 3 — Validador de Senha
Programa responsável por validar senhas com base em regras definidas.

**Conceitos aplicados:**
- Manipulação de strings
- Validações lógicas
- Regras básicas de segurança
- Controle de fluxo

---

### Projeto 4 — Sistema de Cadastro
Sistema simples de cadastro de usuários via terminal.

**Conceitos aplicados:**
- Estruturas de dados
- Organização de informações
- Menus interativos
- Entrada e armazenamento de dados

---

### Projeto 5 — Jogo de Adivinhação
Jogo em que o usuário tenta adivinhar um número gerado aleatoriamente.

**Conceitos aplicados:**
- Laços de repetição
- Comparações
- Controle de tentativas
- Uso do módulo `random`

---

## Evolução e Refatoração

Os projetos representam **versões iniciais (V1)** e serão futuramente **refatorados**, com foco em:

- Melhor organização do código
- Uso mais consistente de funções
- Separação de responsabilidades
- Validações mais robustas
- Aplicação de boas práticas aprendidas ao longo do estudo

## Tecnologias Utilizadas
